# Publicacion Para LinkedIn

## Version Corta

Hoy comparto VeciReport, un sistema web para gestionar reportes vecinales dentro de un fraccionamiento.

El proyecto permite registrar vecinos, aprobar cuentas, levantar reportes de servicios, asignar trabajadores, marcar incidencias como atendidas y administrar fraccionamientos con validacion por mapa.

Stack:

- PHP 8
- MySQL
- JavaScript vanilla
- Leaflet/OpenStreetMap
- CSS responsive
- Sesiones, roles y CSRF

Demo:

```text
https://vecireport.gt.tc/
```

Repositorio:

```text
https://github.com/MauricioMurcia293847/VeciReport
```

## Puntos A Mencionar

- Registro con comprobante de domicilio.
- Validacion de ubicacion dentro del fraccionamiento usando mapa.
- Panel del vecino y panel administrativo.
- Reportes con estados: pendiente, en proceso y atendido.
- Asignacion de trabajadores disponibles.
- Bitacora de acciones del sistema.
- Demo publica con cuentas protegidas.
